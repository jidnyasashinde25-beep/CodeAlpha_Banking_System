# CodeAlpha Banking System

## Description
A simple Banking System developed in C++ using Object-Oriented Programming and File Handling. The system allows users to create bank accounts, deposit and withdraw money, and check account balances through a menu-driven interface.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Menu Driven Program
- File Handling
- Object-Oriented Programming

## Technologies Used
- C++
- OOP Concepts
- File Handling
